# Moment-A

Moment-A es una plataforma interactiva diseñada para conectar usuarios con experiencias, sorteos y comercios locales a través de un mapa dinámico y una interfaz moderna.

## Características Principales

- **Mapa Interactivo**: Visualiza eventos, sorteos y tiendas en tiempo real con marcadores personalizados.
- **Sorteos en Tendencia**: Acceso rápido a los sorteos más populares y exclusivos.
- **Perfiles de Usuario**: Gestión de perfil y seguimiento de interacciones.
- **Mensajería**: Salas de contacto integradas para comunicación directa.
- **Diseño Premium**: Interfaz fluida con efectos de glassmorphism y animaciones modernas.

## Tecnologías Utilizadas

- **Frontend**: Next.js, React, TypeScript.
- **Estilos**: Tailwind CSS / Vanilla CSS.
- **Componentes**: UI personalizada optimizada para rendimiento y estética.

## Instalación y Desarrollo

Para ejecutar el proyecto localmente:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/[TU-USUARIO]/moment-a.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Despliegue

El proyecto está optimizado para ser desplegado en [Vercel](https://vercel.com/new).
